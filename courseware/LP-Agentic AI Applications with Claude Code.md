# Lesson Plan — Agentic AI Applications with Claude Code

**Course Code:** TGS-2025052468  |  **Trainer:** Dr. Alfred Ang  |  **Duration:** 1 Day (9:00 AM – 6:00 PM)

**Mode of Delivery:** Instructor-led, hands-on (lecture, demonstration, practical activities)

---

## Learning Outcomes

By the end of this course, learners will be able to:

1. Explain the agentic loop and apply context-engineering techniques in Claude Code.
2. Create custom slash commands and connect MCP tools to extend Claude Code.
3. Use agent skills, sub agents and hooks to automate specialised tasks.
4. Build, deploy, test, secure and optimise a real website end-to-end with Claude Code.

## Topic-by-Topic Breakdown

### Topic 1 — Claude Code Fundamentals (≈ 1 hr 15 min + Activity 1)

- Setup: Terminal, IDE (VS Code) and Desktop
- How Claude Code works: the agentic loop and built-in tools
- Context engineering & the context window (/context, /compact, /clear)
- Memory files: CLAUDE.md vs auto memory (/init, /memory)
- Permission modes and the dangers of bypass permissions
- Managing sessions, scheduled tasks (Routines), and /goal
- **Activity 1:** Build and deploy a website

### Topic 2 — Tools and Commands (≈ 1 hr 30 min incl. activities)

- Custom slash commands and arguments
- **Activity 2:** Create a /gitpush custom command
- MCP tools and architecture; Playwright MCP
- **Activity 3:** Install Playwright MCP, screenshot site, push
- **Activity 3B:** Test the enquiry form with Playwright; confirm enquiries reach your email

### Topic 3 — Skills, Agents & Hooks (≈ 1 hr 30 min + Mini Capstone)

- Agent skills and installation
- **Activity 4:** Install & run frontend-design and seo-audit skills
- Sub agents and hooks
- **Activity 5:** Security and UI/UX (lead-magnet) sub agents
- **Activity 6 / Mini Capstone:** Form testing, WhatsApp widget, final review and presentation
