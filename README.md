## About the course

A hands-on, one-day WSQ course on building real applications with **Claude Code**. Learners go from a plain-English goal to a deployed app using the agentic loop, context engineering, custom commands, MCP tools, skills, sub agents and hooks — then build and present their own mini-capstone project.

**Topics**

1. **Claude Code Fundamentals** — setup, the agentic loop & tools, context engineering, memory, permissions, sessions, scheduled tasks, `/goal`, and the 7-step build workflow.
2. **Tools and Commands** — the `.claude` directory, custom slash commands, MCP tools and Playwright MCP.
3. **Skills, Agents & Hooks** — agent skills, sub agents, hooks, and a mini-capstone project.
